
** IMPORTANT:
- Stop any HTTP server instances running on your computer. 
  Example- Apache,IIS before you install/start WAMP
  For Windows: Run>services.msc ; Stop Apache, WorldWideWebPublishing
- After Installation, http://localhost should land you in default WAMP/MAMP page
- Apache should listen on port 80 and MySQL on 3306
- Default user for Mysql is 'root' and password is ''
- To change password, Open MySQL console from WAMP/MAMP menu, type:
msql> SET PASSWORD FOR 'root'@'localhost'=PASSWORD('root123');
